# SOC Playbooks

This repository contains Security Operations Center (SOC) playbooks that outline standardized procedures for investigating and responding to common security incidents.

## Objectives

- Standardize incident response workflows
- Improve investigation consistency
- Document containment and remediation procedures
- Map techniques to the MITRE ATT&CK framework
